# Dossier Images

Placez ici toutes les photos du salon.

## Images Requises

### Hero Section
- **hero-salon.jpg** - Photo principale du salon (1920x1080px recommandé)

### Section Salon
- **salon-dames.jpg** - Photo du salon côté dames (800x600px minimum)
- **salon-hommes.jpg** - Photo du salon côté hommes (800x600px minimum)

### Section Services
- **hair-katsura.jpg** - Photo pour le service Hair Katsura (1000x700px recommandé)

### Section Coffrets
- **coffret-rasage.jpg** - Photo du coffret rasage (600x600px)
- **coffret-barbe.jpg** - Photo du coffret barbe (600x600px)
- **produits.jpg** - Photo des produits à l'unité (800x600px)

### Galerie
- **galerie-1.jpg** à **galerie-6.jpg** - Photos de la galerie (minimum 6, format carré 800x800px recommandé)
- Vous pouvez ajouter plus d'images en suivant le pattern `galerie-N.jpg`

## Formats Recommandés

- **Format** : JPG ou WebP (pour meilleure compression)
- **Qualité** : Optimisée pour le web (80-90% pour JPG)
- **Taille** : Maximum 500KB par image
- **Dimensions** : Voir recommandations ci-dessus

## Optimisation

Pour optimiser vos images avant de les ajouter :

1. **Compression** : Utilisez des outils comme TinyPNG, ImageOptim ou Squoosh
2. **Redimensionnement** : Ajustez les dimensions selon les recommandations
3. **Nommage** : Respectez exactement les noms de fichiers indiqués

## Note

Si vous n'avez pas encore toutes les images, le site fonctionnera quand même. Les placeholders CSS afficheront des zones de couleur en attendant.
